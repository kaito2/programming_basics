## 2.1

```ocaml
# 7 - 3 * 4 ;;
- : int = -5
# 7 / 2 * 2 ;;
- : int = 6
# 7 * 2 / 2 ;;
- : int = 7
```

整数型なので除算は切り捨て

## 2.2
